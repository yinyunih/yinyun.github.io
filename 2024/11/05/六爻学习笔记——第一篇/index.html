一、用神、原神、仇神、忌神

在六爻算卦中，用神、原神、仇神、忌神是最基本也是最重要的概念。

1.1 什么是用神以及如何选取用神？

用神的选取主要是取决与你想要占卜的事物。六爻最终通过用神的旺衰来断定事物的吉凶。

  1. 出现以下情况选取父母爻为用神：
    当占卜父母，在父母之上的长辈，父母同辈的亲人及老师时，应当选取卦中的父母爻为用神。占卜天地、城池、宅舍、屋宇、衣服、文书等等也以父母爻为用神。
    关键点为，一切庇覆我的都是。
  2. 出现以下情况选取官鬼爻为用神：
    当占卜功名、官府、鬼神、妻占夫时，都以官鬼爻为用神。占乱臣、盗贼、邪祟也以官鬼爻为用神。
    关键点为，一切拘束我的都是。
  3. 出现以下情况选取兄弟爻为用神：
    当占卜兄弟姐妹，本族中的兄弟、姑姨、姐夫妹夫、以及结拜兄弟，都以兄弟爻为用神。
    占卜表兄弟，以兄弟爻为用神不应验，以应爻为用神。
  4. 出现以下情况选取妻财爻为用神：
    占卜妻妾、婢女、仆役时都以妻财爻为用神。占货物、珠宝、钱财、粮米也用妻财爻为用神。
    关键点为，一切凡属我驱使的人、使用的财物、什物器皿都是
  5. 出现以下情况选取妻财爻为用神：
    占卜子孙、女儿、女婿、外甥、门徒凡在子孙辈中的，都以子孙爻为用神。
    占卜忠趁、良将、医生、医药、僧人、道士、六畜禽兽，都以子孙爻为用神。

1.2 原神、仇神、忌神的概念？

  • 原神：是生用神的爻。
  • 忌神：是克用神的爻。
  • 仇神：是克原神的爻，不生用神，反生忌神的爻。

举例：
当金为用神。
土生金，那么土就是原神。
火克金，那么火就是忌神。
木克土而生火，那么木就是仇神。

二、原神、忌神的旺衰

原神虽然能生用神，但是需要旺相才能去生用神。对于忌神也是一样，旺相才能去克用神。
所谓旺相，打个比方就是一个人有力气才能做一些体力劳动，例如搬砖。对于爻来说，就是自生旺了，有力量了，才能去生别的爻。

2.1 原神能生用神的五种情况：

  1. 原神旺相，或临日辰月建，或者日辰、月建、动爻生扶。

  2. 原神发动化回头生及化进神。

  3. 原神在日辰上长生或帝旺。

  4. 原神与忌神同时发动。

  5. 原神旺相发动、临旬空化旬空。

2.2 原神不能生用神的六种情况：

  1. 原神休囚不动，或者发动而休囚，又被伤克。

  2. 原神休囚，又逢旬空月破。

  3. 原神休囚，发动而化退神。

  4. 原神衰弱而又遇绝。

  5. 原神入三墓。

  6. 原神休囚、发动而化绝、化克、化月破、化散。

2.3 忌神能克用神的五种情况：

  1. 忌神旺相，或临日辰月建，或者日辰、月建、动爻生扶。

  2. 忌神发动化回头生及化进神。

  3. 忌神旺相发动、临旬空化旬空。

  4. 忌神在日辰上长生或帝旺。

  5. 忌神与仇神同时发动。

2.4 忌神不能克用神的七种情况：

  1. 忌神休囚不动，或者发动而休囚，或被日辰、月建、动爻克制。

  2. 忌神休囚，又逢旬空月破。

  3. 忌神入三墓。

  4. 忌神发动而化退神。

  5. 忌神衰弱而又遇绝。

  6. 忌神发动而化绝、化克、化月破、化散。

  7. 忌神与原神同时发动。

以上讨论原神忌神有力无力的情况，也要用神有气。倘若用神无气，那么叫做“原神有力亦难生，忌神无力何足喜。”
